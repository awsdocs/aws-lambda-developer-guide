# FilterCriteria<a name="API_FilterCriteria"></a>

 An object that contains the filters for an event source\.

 ** Filters **   <a name="SSS-Type-FilterCriteria-Filters"></a>

Type: Array of [Filter](API_Filter.md) objects  
Required: No
